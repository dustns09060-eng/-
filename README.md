여우방 V31 통합 버전

기능
1. 팔로우리스트
- Google Sheet Sheet1 연동
- 500명씩 보기
- 검색
- 인스타 바로가기

2. 맞팔확인
- 인스타그램 정보 다운로드 ZIP 파일 그대로 첨부
- ZIP 내부 followers / following HTML 또는 JSON 자동 탐색
- 맞팔 / 나만 팔로우 / 상대만 팔로우 / 방명단만 필터
- 검색 및 인스타 바로가기
- 분석은 사용자의 기기 안에서만 진행

3. 운영진 기능 (비밀번호 0702)
- 구글시트 열기
- 명단 새로고침
- 공지 작성 / 삭제
- 운영 통계

주의
- ZIP 분석은 JSZip CDN을 사용하므로 인터넷 연결이 필요합니다.
- 인스타그램 ZIP 내부 구조가 크게 바뀌면 추가 수정이 필요할 수 있습니다.
